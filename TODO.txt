1. Смена команды
	1.1 Загрузка ответов
	1.2 Загрузка писем
	
2. Переключение режима ALL/NOT-GRADED-TEAMS and vice-versa
	2.1 Загрузка списка команд
	2.2 Загружаем список команд. Если команда, которая была выбрана до смены режима есть в новом списке, то переключаемся на неё. Ответы и письма этой команды уже загружены.
		Если нет - на первую команду из новго списка, и выполняем загрузку ответов и загрузку писем для выбранной команды.
		
3. Проставление оценки команде.
	3.1 Если у ответа была оценка раньще, то просто меняем оценку в записи ответа и ничего не загружаем нового.
		Если у ответа раньше оценки не было, то:
			3.1.1 Удаляем ответ из списка ответов без оценки
			3.1.2 Если у текущей команды не осталось ответов без оценки, проверяем режим отображения
				  Если текущий режим отображения NOT-GRADED-TEAMS
						3.1.2.1 Если есть команды в списке, то переключаемся на другую команду из списка и загружаем ответы и письма для неё.
						3.1.2.2 Если других команд в списке нет, то автоматически переключаемся в режим ALL, 
								загружаем все команды, после чего становимся на команду, которая была выбрана в момент выставления оценки.
			3.1.3 Если остались ответы без оценки - то ничего дополнительно не делаем.
			
4. Удаление письма и связанных с ним ответов.
	4.1 Если включён режим NOT-GRADED-TEAMS, проверяем есть-ли ещё ответы без оценок для текущей команды
		4.1.1 Если нет ответов без оценок для текущей команды, проверяем наличие других команд в списке команд с ответами без оценок.
			4.1.1 Если такие команды есть, переключаемся на первую команду из такого списка и загружаем для неё вопросы и письма.
			4.1.2 Если таких команд нет, то переключаемся в режим ALL и выставляем в качестве команды ту команду, которая была в момент удаления письма с ответами.
				  Ничего загружать не надо, кроме удаления записи о письме и записи об ответах из таблиц на фронте.
