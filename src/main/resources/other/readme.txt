Poetica

Страница проекта: https://github.com/cdefgah/poetica
Пользовательская документация: https://github.com/cdefgah/poetica/wiki

Программа по-умолчанию стартует по порту 7777.

Порт указан в файле config/application.properties. 

После запуска start-poetica.bat (Windows), либо start-poetica.sh (Linux/MacOS),
как только в консоли прекратится движение и строки перестанут появляться, открываем браузер и указываем там адрес:

http://127.0.0.1:7777

Приложение откроется в браузере.

В папке initial-data в текстовом файле находится список команд, принимающих участие в чемпионате по бескрылкам.

Его можно импортировать в программу со страницы "Команды", нажав соответствующую кнопку вверху справа.
